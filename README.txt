## About
2d top down shooter with tactical gameplay set in a procedurally generated world.

## Controls
WASD: move
Mouse: look
Lmb: shoot
Rmb: walk to position
R: reload
Q: use left Gadget
E: use right Gadget
F: pick up item or swap Gadgets
Esc: exit

## Dependencies
- SFML
- Thor
- yaml-cpp
- TGUI
- Let There Be Light
