## About
2d top down shooter with tactical gameplay set in a procedurally generated world.

## Controls
WASD: walk
Mouse: aim
Left Mouse Button: shoot
R: reload
Q: use left gadget
E: use right gadget
F: pick up item or swap gadgets
Esc: exit game

## Dependencies
- SFML
- Thor
- yaml-cpp
- TGUI
- Let There Be Light
